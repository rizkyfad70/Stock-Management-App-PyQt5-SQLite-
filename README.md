# Stock-Management-App-PyQt5-SQLite-
Maintenance Stock Management App is an offline desktop application built using Python, PyQt5, and SQLite3. The app is designed to help technicians and maintenance teams manage part inventories efficiently, record stock in/out activities, and monitor stock levels in real-time.
